2 Connect email accounts

Prerequisite You should already have 5 secondary domains with Google Workspace, SPF, DKIM, and DMARC configured. See config/domain-setup.md.

Connect to Instantly

  1. Go to Email Accounts → Add New → Google Workspace
  2. Sign in with each sending account (e.g., rok@trynevron.com)
  3. Repeat for all accounts across all 5 domains
  4. For each account, set:
    • From name: Rok from Nevron or Rok Kokalj | Nevron
    • Reply-to: rok.kokalj@nevron.eu
    • Daily sending limit: 30

Do this immediately after connecting accounts New domains need 3-4 weeks of warmup before any campaign sends. Do not skip this.

Warmup timeline

Week 1-2: Warmup only

No campaign sends. Accounts building reputation in the warmup network.

Week 3: Health check

Check health scores for each account. Need 90%+ to proceed.

Week 4: First campaign launch

If healthy, start first campaign with slow ramp enabled.

Never turn warmup off Keep it running alongside active campaigns, indefinitely.

4 Custom tracking domain

Optional but recommended. Isolates your tracking reputation from Instantly's shared domain.

  1. Pick a subdomain: t.trynevron.com
  2. Add CNAME record in DNS:
    t.trynevron.com → prox.itrackly.com
  3. In Instantly: Settings → Tracking Domain → add custom domain
  4. Enable "Warm Custom Tracking Domain" in warmup settings

6 Lead import

Recommended flow (Phase 1)

Apollo → apollo_attio_sync.py → Attio (scored + enriched) → CSV export → Instantly
  1. Run apollo_attio_sync.py --market FL to enrich and score leads in Attio
  2. In Attio, filter: Market = Florida, ICP Tier = 1 or 2
  3. Export filtered leads as CSV
  4. In Instantly campaign → Leads → Upload CSV
  5. Map CSV columns to Instantly fields
  6. Repeat per market

7 Attio integration

Three options, from simplest to most automated. Start with Option C, upgrade to A when volume justifies it.

Option C: Manual sync (launch with this)

  1. Check Instantly Unibox daily for replies
  2. When lead replies: update deal stage in Attio manually (Lead → Replied)
  3. When meeting booked: update to Demo Booked in Attio
  4. Upgrade to Zapier once sending 100+ emails/day

Option A: Zapier ($20+/month)

ZapTriggerAction
Reply → Attio Instantly: New Event (reply_received) Attio: Update List Entry → stage "Replied"
Sent → Attio Instantly: New Event (email_sent, first only) Attio: Update List Entry → stage "Sequence Active"
Meeting → Attio Instantly: New Event (meeting_booked) Attio: Update List Entry → stage "Demo Booked"
Disqualified → Block Attio: Stage = "Disqualified" Instantly: Add to Block List

Option B: Webhooks + custom script (no Zapier cost)

  1. In Instantly: Settings → Integrations → Webhooks
  2. Add endpoint URL (your server or serverless function)
  3. Select events: reply_received, email_sent, meeting_booked, bounced, unsubscribed
  4. Build receiver that calls Attio API to update deal stages
Requires Hypergrowth plan Webhooks are not available on the Growth plan.

10 Launch sequence

Week 1-4: Warmup (all accounts)

Connect accounts, enable warmup, configure DNS. No campaign sends.

Week 4-5: Florida launch

Start with 1 campaign (Florida Renovation). 20-30 Tier 1 leads. Slow ramp ON. Monitor 1 week, then expand.

Week 6-7: Netherlands launch

Same ramp pattern. Europe/Amsterdam timezone. English sequences.

Week 8-10: Slovenia + Croatia

Local language sequences. Smaller markets, can launch simultaneously. Extra GDPR compliance checks.

Scaling rules Never launch more than 2 new campaigns in the same week. Never add more than 50 new leads per campaign per day. Keep total daily volume under 80% of theoretical max. When in doubt, slow down -- reputation is easier to maintain than rebuild.

11 Compliance checklist

CAN-SPAM (Florida campaigns)

  • Physical address in email signature (Nevron business address)
  • Unsubscribe link/header enabled in Instantly
  • Honor unsubscribes within 10 business days (Instantly auto-handles)
  • No deceptive subject lines
  • Identify message as advertisement (first email)

GDPR (Netherlands, Slovenia, Croatia)

  • Legitimate interest basis documented for B2B outreach
  • Business email addresses only (no personal emails)
  • Easy opt-out in every email
  • Data processing records maintained
  • Right to erasure: on request, block in Instantly + delete from Attio
  • Company registration info in EU email signatures

12 API setup (for future automation)

Get API key

  1. Settings → Integrations → API
  2. Create new V2 API key
  3. Name: nevron-sdr-sync
  4. Scopes: Campaigns (R+W), Leads (R+W), Analytics (R)
  5. Save key to .env

Add to .env

# Add to scripts/.env alongside Apollo and Attio keys
INSTANTLY_API_KEY=your_instantly_api_v2_key_here
