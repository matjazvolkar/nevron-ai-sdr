LinkedIn Commenting Strategy

💬

The 3C Framework

Compliment something specific. Contribute your own experience or data. Continue with a thoughtful question. Comments over 15 words get 2.5x more algorithmic weight.

📊

Daily Activity

10-15 meaningful comments/day on ICP prospects' and industry leaders' posts. Never pitch in comments. Mix automated viewing with genuine manual engagement.

🔄

Warm-Up Funnel

Week 1-2: View profiles, like, comment. Week 2-3: Send connection request referencing their content. After connection: Continue engaging, then personalized DM.

LinkedIn Limits & Risks
LinkedIn allows 100 connection invites/week (all account types). Stay at 70-80% of limits. Randomize timing. LinkedIn explicitly prohibits automation tools, so use tools that mimic human behavior and stay conservative. Consequences range from temporary blocks to permanent restriction. Always warm up accounts gradually.

Email Copy: What Works in 2025/2026

📝

Format

6-8 sentences, under 200 words. No HTML, no images, no fancy formatting. Plain text performs best. Soft CTAs ("How are you thinking about this?") outperform hard asks.

🔄

Sequence

2-3 emails total. Each adds new value. Day 0: personalized hook. Day 3-4: new angle/case study. Day 7-10: graceful breakup. Long sequences trigger spam complaints.

🎯

Personalization

Reference a specific trigger (renovation, expansion, new GM, rating change). Lead with their world, not your product. AI drafts + human refinement in 10-15 seconds per email.

Nevron Email Angle Examples
Trigger-based: "Noticed [Hotel] just completed a renovation. When properties modernize the physical experience but keep legacy tech for guest interactions, there's usually a gap. Curious how you're thinking about the digital side?"

Rating-based: "Looked at [Hotel]'s recent TripAdvisor reviews. Several mention 'great service but outdated TV system.' Our data shows a 0.1 higher OTA score translates to ~EUR 10 more per room/night. Worth a 10-minute conversation?"

Competitor-based: "I see [Hotel] is using [Competitor IPTV]. We've migrated 300+ properties from similar systems. The most common feedback: 'wish we'd switched sooner.' What's your experience been?"

Implementation

Phased Rollout Plan

Start with the foundation, prove it works, then upgrade. Each phase builds on the previous one. Don't skip steps.

Phase 1: Foundation (Weeks 1-4)
Infrastructure + First Campaigns
Get the basics right. No outreach until infrastructure is solid.
  • ICP Definition: Finalize target criteria with marketing manager (hotel size, star rating, geography, decision maker titles)
  • Email Infrastructure: Buy 3-5 secondary domains, set up SPF/DKIM/DMARC, create 3 mailboxes per domain on Google Workspace
  • Warmup: Start automated warmup on all mailboxes (Instantly or Smartlead). Minimum 2 weeks before sending.
  • Lead Sourcing: Use Apollo (free/basic) + LinkedIn Sales Navigator to build first 200-lead list
  • CRM Setup: Set up Attio (free) or evaluate Odoo CRM capabilities
  • LinkedIn Profile: Optimize Nevron SDR's LinkedIn profile for hospitality outreach
  • First Campaigns: 3-4 email campaigns, 50 recipients each, testing different angles (renovation trigger, rating-based, AI transformation, competitor migration)
  • Budget estimate: EUR 200-400/mo (domains, Apollo, Instantly basic, Sales Navigator)
Phase 2: Optimize + LinkedIn (Weeks 5-8)
Add LinkedIn, Test Messaging
Based on Phase 1 data, refine messaging and add the second channel.
  • Analyze: Which email angle got the best reply rate? Which ICP segment responded?
  • A/B Test: Subject lines, opening lines, CTAs. Test one variable at a time, 100+ sends per variant.
  • LinkedIn Launch: Start commenting strategy (10-15/day on ICP prospects). Begin connection request outreach.
  • Test LinkedIn Tools: Try Claude in Chrome MCP for automated engagement. Compare with PhantomBuster or Expandi.
  • Enrichment: If Apollo's data isn't sufficient, add Clay for waterfall enrichment
  • Templates: Document winning email templates and LinkedIn scripts
  • Budget estimate: EUR 300-600/mo (add LinkedIn tool + potentially Clay)
Phase 3: Multi-Channel (Weeks 9-12)
Coordinated Sequences + Personalization
Combine email + LinkedIn into coordinated sequences. Layer in AI personalization.
  • Multi-Channel Sequencing: Implement coordinated email + LinkedIn sequences (La Growth Machine or manual coordination via Instantly + Expandi)
  • AI Personalization: Use Clay + Claude to generate personalized opening lines at scale. Scrape hotel websites and reviews for hooks.
  • Signal-Based Triggers: Monitor for renovation announcements, new GM hires, funding events in hospitality
  • Scale Volume: Add more sending domains and mailboxes. Target 200-500 outreach touches per week.
  • Meeting Booking: Set up Calendly or similar for easy scheduling from email/LinkedIn CTAs
  • Budget estimate: EUR 500-1,000/mo
Phase 4: Scale & Automate (Month 4+)
AI SDR + Full Automation
Once the playbook is proven, automate and scale.
  • Custom AI SDR: Build Claude Code + MCP automation for prospecting, enrichment, and email personalization
  • Or Evaluate SaaS AI SDR: AiSDR ($900/mo) or Jason AI ($500/mo) for supplemental automation
  • Expand Geography: Roll out to new regions (Austria, Northern Italy, Gulf) with localized messaging
  • Intent Data: Add Bombora or similar for topic-based intent signals (hotels researching AI, guest experience tech)
  • Referral Engine: Build automated referral requests from happy customers (Nevron already has a referral program)
  • Budget estimate: EUR 1,000-3,000/mo

Troubleshooting

"401 Unauthorized" from Apollo

  • Your API key is wrong or expired
  • Go to Apollo -> Settings -> Integrations -> API -> regenerate key
  • Update `.env`

"401 Unauthorized" from Attio

  • Your API token is wrong or expired
  • Go to Attio -> Settings -> Developers -> API keys -> create new key
  • Update `.env`

"422 Unprocessable Entity" from Attio

  • A field slug doesn't match or a select option doesn't exist
  • Run `--discover` to check actual slugs
  • Make sure all select options in Attio match exactly what the script sends (e.g., `Florida` not `FL`)

"429 Too Many Requests" from Apollo

  • You've hit the rate limit
  • The script handles this with automatic backoff
  • If it keeps happening, reduce the `--limit` or wait a few minutes

Duplicate leads appearing in Attio

  • The dedup check failedusually means the domain or email format doesn't match
  • Check that Apollo's `website_url` matches how Attio stores domains (e.g., `palmbeachhotel.com` not `https://www.palmbeachhotel.com`)

Script crashes mid-sync

  • Check the log file in `logs/` for the error
  • Re-run with the same `--market` flagthe dedup logic will skip already-synced leads
  • The script is safe to re-run (idempotent)

A lead was synced that shouldn't have been

  • In Attio, move the Deal to "Disqualified" and fill the `Disqualifier` field with the reason
  • Add the chain name or keyword to the script's `CHAIN_KEYWORDS` list to prevent future matches

def apollo_search(market_code: str, page: int = 1, per_page: int = 100) -> dict:
    """Search people in Apollo for a specific market."""
    market = MARKETS[market_code]

    payload = {
        "page": page,
        "per_page": per_page,
        "person_titles": [
            "General Manager", "Owner", "Managing Director",
            "Hotel Director", "Owner-Operator",
        ],
        "person_seniorities": ["director", "vp", "c_suite", "owner"],
        "organization_industry_tag_ids": ["hospitality"],
        "organization_num_employees_ranges": ["51,200"],
        "person_locations": [market["location"]],
        "contact_email_status": ["verified"],
    }

    for attempt in range(1, APOLLO_MAX_RETRIES + 1):
        resp = requests.post(
            f"{APOLLO_BASE}/mixed_people/search",
            headers=apollo_headers(),
            json=payload,
            timeout=30,
        )

        if resp.status_code == 200:
            return resp.json()
        elif resp.status_code == 429:
            wait = APOLLO_BACKOFF_SECONDS * attempt
            logging.getLogger("apollo_attio").warning(
                f"Apollo rate limit hit. Waiting {wait}s (attempt {attempt}/{APOLLO_MAX_RETRIES})"
            )
            time.sleep(wait)
        else:
            resp.raise_for_status()

    raise Exception(f"Apollo API failed after {APOLLO_MAX_RETRIES} retries")

def validate_icp(person: dict, org: dict) -> tuple[bool, str]:
    """
    Validate a lead against ICP hard filters.
    Returns (passed: bool, reason: str).
    """
    if not org:
        return False, "no_organization"

    # Check chain keywords in company name
    org_name = (org.get("name") or "").lower()
    for chain in CHAIN_KEYWORDS:
        if chain in org_name:
            return False, f"chain_detected:{chain}"

    # Check disqualify keywords in company name and keywords
    org_keywords = " ".join(org.get("keywords") or []).lower()
    combined = f"{org_name} {org_keywords}"
    for kw in DISQUALIFY_KEYWORDS:
        if kw in combined:
            return False, f"disqualified_segment:{kw}"

    # Check email is verified
    email_status = person.get("email_status") or ""
    if email_status != "verified":
        return False, f"email_not_verified:{email_status}"

    # Check person has an ICP-relevant title
    title = (person.get("title") or "").lower()
    if not any(t in title for t in ICP_TITLES):
        return False, f"title_not_icp:{person.get('title', 'none')}"

    # Check person has an email
    if not person.get("email"):
        return False, "no_email"

    return True, "passed"

def calculate_score(person: dict, org: dict, market_code: str) -> dict:
    """
    Calculate partial ICP score from Apollo data.
    Returns dict with score breakdown.
    """
    market = MARKETS[market_code]
    title = (person.get("title") or "").lower()
    org_name = (org.get("name") or "").lower()
    org_keywords = " ".join(org.get("keywords") or []).lower()
    combined = f"{org_name} {org_keywords}"

    scores = {}

    # Room countcan't determine from Apollo, default 2
    scores["room_count"] = 2

    # Segmentfrom keywords
    scores["segment"] = 1
    for points, keywords in SEGMENT_KEYWORDS.items():
        if any(kw in combined for kw in keywords):
            scores["segment"] = points
            break

    # Geography
    scores["geography"] = market["geo_score"]

    # Decision-maker access
    if any(kw in title for kw in DECISION_MAKER_KEYWORDS):
        scores["decision_maker"] = 3
    elif any(kw in title for kw in ["director", "vp", "vice president"]):
        scores["decision_maker"] = 2
    else:
        scores["decision_maker"] = 1

    # Tech signalscan't determine from Apollo, default 2
    scores["tech_signals"] = 2

    # Trigger eventscan't determine from Apollo, default 1
    scores["trigger_events"] = 1

    # Review scorecan't determine from Apollo, default 2
    scores["review_score"] = 2

    scores["total"] = sum(scores.values())

    # Tier assignment
    total = scores["total"]
    if total >= 12:
        scores["tier"] = "Tier 1"
    elif total >= 9:
        scores["tier"] = "Tier 2"
    else:
        scores["tier"] = "Tier 3"

    return scores

def normalize_title(raw_title: str) -> str:
    """Map Apollo title to Attio select option."""
    t = raw_title.lower()
    if any(kw in t for kw in ["general manager", " gm", "gm "]):
        return "GM"
    if any(kw in t for kw in ["owner", "proprietor", "founder"]):
        return "Owner-Operator"
    if "managing director" in t or "director general" in t:
        return "Managing Director"
    if "front office" in t:
        return "Front Office Manager"
    if "revenue" in t:
        return "Revenue Manager"
    return "Other"

def extract_domain(url: str) -> str:
    """Extract clean domain from a URL."""
    if not url:
        return ""
    domain = url.lower().strip()
    for prefix in ["https://", "http://", "www."]:
        if domain.startswith(prefix):
            domain = domain[len(prefix):]
    domain = domain.rstrip("/")
    return domain

def build_company_values(org: dict, market_code: str, scores: dict) -> dict:
    """Build Attio Company record values from Apollo org data."""
    market = MARKETS[market_code]
    domain = extract_domain(org.get("website_url", ""))
    slugs = ATTIO_COMPANY_SLUGS

    values = {
        "name": [{"value": org.get("name", "")}],
    }

    if domain:
        values["domains"] = [{"domain": domain}]

    # Custom fieldsselect/number types
    values[slugs["market"]] = market["name"]
    values[slugs["icp_score"]] = scores["total"]
    values[slugs["icp_tier"]] = scores["tier"]
    values[slugs["outreach_angle"]] = market["default_angle"]

    # Segmenttry to detect from keywords
    org_keywords = " ".join(org.get("keywords") or []).lower()
    org_name = (org.get("name") or "").lower()
    combined = f"{org_name} {org_keywords}"

    for points, keywords in SEGMENT_KEYWORDS.items():
        for kw in keywords:
            if kw in combined:
                # Map to Attio select option
                segment_map = {
                    3: "Luxury",  # luxury, wellness, etc.
                    2: "Upscale",  # upscale, lifestyle, boutique
                    1: "Upper-Midscale",
                }
                values[slugs["segment"]] = segment_map.get(points, "Upscale")
                break

    return values


def build_person_values(person: dict, market_code: str) -> dict:
    """Build Attio Person record values from Apollo person data."""
    market = MARKETS[market_code]

        "last_name": [{"last_name": person.get("last_name", "")}],

    # LinkedIn URL
    linkedin = person.get("linkedin_url", "")

    return values

def sync_lead(
    person: dict,
    org: dict,
    market_code: str,
    dry_run: bool,
    log: logging.Logger,
    stats: dict,
) -> None:
    """Process a single lead: validate, score, push to Attio."""
    name = f"{person.get('first_name', '?')} {person.get('last_name', '?')}"
    title = person.get("title", "?")
    org_name = org.get("name", "?") if org else "?"
    domain = extract_domain(org.get("website_url", "")) if org else ""
    email = person.get("email", "?")

    log.debug(f"Processing: {name}{title}{org_name}")

    # 1. Validate ICP
    passed, reason = validate_icp(person, org)
    if not passed:
        log.info(f"  SKIP {name} ({org_name}){reason}")
        stats["disqualified"] += 1
        stats["disqualify_reasons"][reason] = stats["disqualify_reasons"].get(reason, 0) + 1
        return

    stats["passed_icp"] += 1

    # 2. Score
    scores = calculate_score(person, org, market_code)
    log.debug(f"  Score: {scores['total']}/21 ({scores['tier']})")

    # 3. Dry runjust print
    if dry_run:
        log.info(
            f"  PASS {name}{title}{org_name} ({domain})\n"
            f"       Score: {scores['total']} | {scores['tier']} | "
            f"Angle: {MARKETS[market_code]['default_angle']} | "
            f"Seq: {market_code}-{MARKETS[market_code]['default_angle']}\n"
            f"       -> Would CREATE company \"{org_name}\"\n"
            f"       -> Would CREATE person \"{name}\" ({email})\n"
            f"       -> Would CREATE deal (stage: Lead)"
        )
        stats["would_create_companies"] += 1
        stats["would_create_people"] += 1
        stats["would_create_deals"] += 1
        return

    # 4. Upsert Company
    try:
        company_values = build_company_values(org, market_code, scores)
        company_result = attio_upsert_company(domain, company_values)
        company_id = company_result.get("id", {}).get("record_id", "")

        if company_result.get("created"):
            log.info(f"  CREATED company: {org_name} ({domain})")
            stats["companies_created"] += 1
        else:
            log.info(f"  UPDATED company: {org_name} ({domain})")
            stats["companies_updated"] += 1
    except requests.HTTPError as e:
        log.error(f"  ERROR creating company {org_name}: {e.response.status_code}{e.response.text}")
        stats["errors"] += 1
        return

    # 5. Upsert Person
    try:
        person_values = build_person_values(person, market_code)
        person_result = attio_upsert_person(email, person_values)
        person_id = person_result.get("id", {}).get("record_id", "")

        if person_result.get("created"):
            log.info(f"  CREATED person: {name} ({email})")
            stats["people_created"] += 1
        else:
            log.info(f"  UPDATED person: {name} ({email})")
            stats["people_updated"] += 1
    except requests.HTTPError as e:
        log.error(f"  ERROR creating person {name}: {e.response.status_code}{e.response.text}")
        stats["errors"] += 1
        return

    # 6. Check for existing deal
    try:
        existing_deals = attio_query_list_entries(ATTIO_LIST_ID, company_id)
        if existing_deals:
            log.info(f"  SKIP dealalready exists for {org_name}")
            stats["deals_skipped"] += 1
            return
    except requests.HTTPError:
        # If query fails, try creating anyway
        pass

    # 7. Create Deal
    try:
        deal_values = build_deal_values(company_id, person_id, market_code, scores)
        attio_create_list_entry(ATTIO_LIST_ID, deal_values)
        log.info(f"  CREATED deal: {org_name} (stage: Lead, seq: {market_code}-{MARKETS[market_code]['default_angle']})")
        stats["deals_created"] += 1
    except requests.HTTPError as e:
        log.error(f"  ERROR creating deal for {org_name}: {e.response.status_code}{e.response.text}")
        stats["errors"] += 1

def sync_market(market_code: str, dry_run: bool, limit: int | None) -> dict:
    """Sync all leads from one market."""
    market = MARKETS[market_code]
    log = setup_logging(market_code, dry_run)

    if dry_run:
        log.info(f"{'=' * 60}")
        log.info(f"DRY RUN MODEno records will be created in Attio")
        log.info(f"{'=' * 60}")
    else:
        log.info(f"{'=' * 60}")
        log.info(f"SYNC MODErecords WILL be created/updated in Attio")
        log.info(f"{'=' * 60}")

    log.info(f"Market: {market['name']} ({market_code})")
    log.info(f"Default angle: {market['default_angle']}")
    if limit:
        log.info(f"Limit: {limit} leads")
    log.info("")

    stats = {
        "total_fetched": 0,
        "passed_icp": 0,
        "disqualified": 0,
        "disqualify_reasons": {},
        "companies_created": 0,
        "companies_updated": 0,
        "people_created": 0,
        "people_updated": 0,
        "deals_created": 0,
        "deals_skipped": 0,
        "errors": 0,
        # Dry run counters
        "would_create_companies": 0,
        "would_create_people": 0,
        "would_create_deals": 0,
    }

    page = 1
    processed = 0

    while True:
        # Fetch page from Apollo
        log.debug(f"Fetching Apollo page {page}...")
        try:
            result = apollo_search(market_code, page=page)
        except Exception as e:
            log.error(f"Apollo search failed: {e}")
            break

        people = result.get("people") or result.get("contacts") or []
        total_entries = result.get("pagination", {}).get("total_entries", 0)
        total_pages = result.get("pagination", {}).get("total_pages", 1)

        if page == 1:
            log.info(f"Apollo search returned: {total_entries} total leads ({total_pages} pages)")
            log.info("")

        if not people:
            break

        for person_data in people:
            if limit and processed >= limit:
                break

            org_data = person_data.get("organization") or {}
            stats["total_fetched"] += 1
            processed += 1

            sync_lead(person_data, org_data, market_code, dry_run, log, stats)

            # Rate limiting for Attio calls (only in live mode)
            if not dry_run:
                time.sleep(0.2)

        if limit and processed >= limit:
            break

        if page >= total_pages:
            break

        page += 1
        time.sleep(APOLLO_DELAY_SECONDS)

    # Print summary
    log.info("")
    log.info(f"{'=' * 60}")
    log.info(f"SYNC SUMMARY{market['name']} ({market_code})")
    log.info(f"{'=' * 60}")
    log.info(f"Total fetched from Apollo:  {stats['total_fetched']}")
    log.info(f"Passed ICP validation:      {stats['passed_icp']}")
    log.info(f"Disqualified:               {stats['disqualified']}")

    if stats["disqualify_reasons"]:
        log.info(f"  Reasons:")
        for reason, count in sorted(stats["disqualify_reasons"].items(), key=lambda x: -x[1]):
            log.info(f"    {reason}: {count}")

    if dry_run:
        log.info(f"Would create companies:     {stats['would_create_companies']}")
        log.info(f"Would create people:        {stats['would_create_people']}")
        log.info(f"Would create deals:         {stats['would_create_deals']}")
    else:
        log.info(f"Companies created:          {stats['companies_created']}")
        log.info(f"Companies updated:          {stats['companies_updated']}")
        log.info(f"People created:             {stats['people_created']}")
        log.info(f"People updated:             {stats['people_updated']}")
        log.info(f"Deals created:              {stats['deals_created']}")
        log.info(f"Deals skipped (existing):   {stats['deals_skipped']}")
        log.info(f"Errors:                     {stats['errors']}")

    log.info(f"{'=' * 60}")
    return stats
